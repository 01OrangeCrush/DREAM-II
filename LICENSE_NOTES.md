# License and Rights Notes

No single repository-wide license is asserted here because the project combines original documentation, research instruments, software or scripts, and metadata describing externally hosted instructional artifacts with different licenses.

Before publishing a formal `LICENSE` file, the authors should select and document licenses separately for:

- original code and analysis scripts;
- original documentation and research instruments;
- released datasets;
- figures and manuscript supplements.

External assignment prompts retain their original copyrights and licenses. Many are linked rather than reproduced. A source license does not automatically authorize republication in this repository.

Recommended practice:

- use metadata and source URLs for restricted artifacts;
- reproduce only materials whose licenses permit redistribution;
- retain attribution and license notices;
- mark third-party material clearly;
- consult the publisher agreement before posting manuscript versions.
